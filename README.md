# log-forwarder

[![Build](https://github.com/gsmlg-ci/log-forwarder/actions/workflows/build.yml/badge.svg)](https://github.com/gsmlg-ci/log-forwarder/actions/workflows/build.yml)

Docker image published to:
- `docker.io/gsmlg/log-forwarder`
- `ghcr.io/gsmlg-dev/log-forwarder`

## Usage

```bash
# From Docker Hub
docker pull gsmlg/log-forwarder:latest

# From GitHub Container Registry
docker pull ghcr.io/gsmlg-dev/log-forwarder:latest
```

## Build

```bash
docker build -t gsmlg/log-forwarder .
```
